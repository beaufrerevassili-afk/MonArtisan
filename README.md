# Application Artisans

## Objectif
Application mobile et web de mise en relation clients/artisans avec ERP intégré.

## Stack
- Frontend Mobile: React Native
- Frontend Web: React.js + Tailwind
- Backend: Node.js + Express
- Base de données: PostgreSQL + Firebase
- Auth: JWT
- Paiements: Stripe + PayPal
- Notifications: Twilio
- Signature électronique: Yousign / DocuSign

## Structure
- frontend_mobile: application mobile iOS/Android
- frontend_web: dashboard web
- backend: API et logique serveur
- docs: documentation et spécifications
